# Voo-flutter

flutter版本的UI库

## Getting Started

Web链接：https://voo-flutter.github.io/#/

## 操作记录

- 本地厂库指向Code平台
- Code平台绑定GitHub平台
    - 每次提交至Code平台，同时同步至GitHub

- GitHub创建Actions
    - 执行自定义的脚本(.github/workflows/main.yml)
    - 脚本打正式web版本代码，并提交至Github Pages工程用于展示

- Github Pages工程
    - 路径：https://github.com/voo-flutter/voo-flutter.github.io.git
    - 工程名必须以github.io结束
